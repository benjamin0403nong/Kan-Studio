# Kan Studio

Kan Studio - 創意設計工作室官方網站

## 網站特色

- 🎨 現代化設計
- 📱 響應式佈局
- ✨ 流暢的動畫效果
- 🚀 簡潔直觀的使用者介面

## 檔案結構

```
kan-studio/
├── index.html          # 主主頁面
├── css/
│   └── style.css      # 樣式表
├── js/
│   └── main.js        # JavaScript 功能
└── images/            # 圖片資料夾
```

## 如何使用

1. 直接開啟 `index.html` 檔案即可在瀏覽器中查看

2. 或使用本地伺服器：
   ```bash
   # Python
   python -m http.server 8000

   # Node.js
   npx http-server
   ```

3. 開啟瀏覽器訪問 `http://localhost:8000`

## 功能區塊

- **首頁**: 吸引人的主視覺設計
- **關於我們**: 介紹工作室理念
- **服務**: 展示提供的服務項目
- **作品集**: 展示過往專案
- **聯絡我們**: 聯絡表單

## 自訂化

- 在 `css/style.css` 中修改顏色和樣式
- 在 `index.html` 中修改內容
- 在 `js/main.js` 中添加互動功能
